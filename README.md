# JetsonHacks NemoClaw Thor

Thor-oriented transition notes and historical helper material for NVIDIA NemoClaw.

This repository is **archived** and is **no longer the recommended install path** for NemoClaw.

For current installation and onboarding, use the upstream NVIDIA/NemoClaw project:

- NVIDIA/NemoClaw: <https://github.com/NVIDIA/NemoClaw>
- NemoClaw documentation: <https://docs.nvidia.com/nemoclaw/>

## Why This Repository Is Archived

This repository was created during an earlier phase of bringing NemoClaw to NVIDIA ARM/NVIDIA GPU systems related to the Jetson and DGX Spark class of machines.

At that time, platform-specific setup, workarounds, and local inference launch helpers were useful. Since then, the upstream NemoClaw project has evolved, and the standard upstream install path now covers the primary installation and onboarding workflow.

As a result, this repository no longer provides enough distinct value to serve as a maintained JetsonHacks install path.

## What This Repository Contains

This repository is retained as a historical reference for:

- earlier Thor-oriented experiments
- local vLLM / Nemotron helper material
- notes from the period when platform-specific installation support was still in flux

These materials may still be useful for reference, but they should not be treated as the primary or current way to install NemoClaw.

## What You Should Do Instead

For a new installation:

1. Go to the upstream NVIDIA/NemoClaw repository
2. Follow the current installation instructions there
3. Use the official NemoClaw documentation for onboarding, providers, and troubleshooting

Start here:

- <https://github.com/NVIDIA/NemoClaw>
- <https://docs.nvidia.com/nemoclaw/>
